## 依赖环境
- [Node.js v12+](https://nodejs.org/)
- [Yarn 1 (Classic)](https://classic.yarnpkg.com/)（可选，目前项目正在使用中）
- [pnpm](https://pnpm.io/)（可选）

## 网站框架
- [VuePress next](https://v2.vuepress.vuejs.org/)

## 安装项目的所有依赖项
```
yarn
```

## 文档预览
```
yarn docs:dev
```

## 文档生成
```
yarn docs:build
```
